Web Teaching Outline

网页入门经验分享 to 特特

两位“老师”

  • 主线学习网址：
    https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web

    简记： https://mdn.io

  • 网上课程：慕课网

    优点：online写代码，不需要编辑器，并且大部分例子都有提供好的模板和代码提示
    缺点：不易掌握学习重点；用户体验…

必需品

编辑器

  • Dreamweaver （几年前用过..）
  • Atom（大神推荐Atom）
  • Brackets （室友墙裂推荐，据说可以实时显示编写效果）

浏览器

  • Chrome
  • Firefox

简单讲解

  1. html代码结构展示

    • html标签 http://www.w3school.com.cn/html/html_basic.asp （这个网址可以当工具书）
    • div布局
      找到“div定位.html”文件，讲解

  2. html文件结构展示

    SIM CITY v3.0文件夹

  3. 移动政务材料展示（ppt, example）

    txt转换成html编写代码，运行后讲解浏览器的功能

  4. css简单讲解

  5. javascript简单了解

自由发挥的经验分享

  • html的内容不需要死记，必要的时候查询就可以
  • html和css没有什么是特别难的，如果有（比如那种看半个小时还看不懂的），那就跳过，它一定不是你现在需要掌握的
  • 一定不要几个小时都在看文档，要看一点写一点，用成就感驱动学习
  • node.js 会写网页的时候再了解
  • markdown 写博客经常用到的语法
